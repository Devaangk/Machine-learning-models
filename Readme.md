# MNIST CNN Digit Classifier

A Convolutional Neural Network (CNN) model built using TensorFlow and Keras for handwritten digit classification on the MNIST dataset.

---

## Features

- CNN-based handwritten digit recognition
- Trained on MNIST dataset
- Accuracy visualization
- Sample image visualization
- Model saving support

---

## Technologies Used

- Python
- TensorFlow / Keras
- Pandas
- Matplotlib
- NumPy

---

## Dataset

MNIST handwritten digit dataset:

- 60,000 training images
- 10,000 testing images
- 10 digit classes (0–9)
